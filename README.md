# Portafolio profesional - Valeria Baltazar Zapata

Portafolio web desarrollado con HTML, CSS y JavaScript.

## Contenido

- `index.html`: página principal.
- `cv.html`: CV web imprimible.
- `css/styles.css`: estilos responsive, modo claro y modo oscuro.
- `js/script.js`: carga automática de repositorios desde GitHub.
- `assets/`: foto profesional, favicon e imágenes.

## Importante

Esta versión no utiliza Font Awesome ni librerías de iconos externas. Los íconos principales se implementan con SVG inline y los íconos de tecnologías se muestran como badges personalizados.
